Secure your account with Multi-Factor Authentication.

API Endpoints

  • POST /auth/register/ Register a new user account.
  • POST /auth/login/ Login and receive JWT access/refresh tokens.
  • GET /auth/mfa/setup/ Get QR code for MFA setup (authenticated).
  • POST /auth/mfa/verify/ Verify MFA token (TOTP or OTP).
  • POST /auth/mfa/send-otp/ Send OTP to user's email (authenticated).
  • POST /auth/token/refresh/ Refresh JWT access token using refresh token.
  • POST /auth/logout/ Logout the current user (authenticated).
